# Changelog

## v1.0.0 — 专业八字 & 八宅风水

### 八字排盘 (src/lib/divination/bazi.ts)
- 真实四柱计算：年柱/月柱（含节气分界）/日柱/时柱
- 十神计算系统：以日主为中心推算十神
- 60甲子纳音五行完整表
- 大运排法：按性别和年柱阴阳顺逆排布
- 喜用神分析：日主强弱判定、喜忌推导
- 日主性格：仁木/礼火/信土/义金/智水五型分析

### 八宅风水 (src/lib/divination/fengshui.ts)
- 命卦计算：男命 `(100-年份后两位)%9`，女命 `(年份后两位-4)%9`
- 东四命/西四命分类
- 八方吉凶：伏位/生气/延年/天医/绝命/五鬼/六煞/祸害
- 房间类型专项建议：卧室/客厅/书房/厨房
- 五行色彩搭配与调和方案

### API 接口
- `POST /api/interpret` — 标准非流式接口
  - `type=bazi`：返回基于真实排盘的专业八字解读
  - `type=fengshui`：返回基于八宅理论的专业风水分析
  - `type=daily`：返回每日运势解读
- 本地模式：API Key 未配置时，全部走本地算法生成
- 真实 AI 模式：配置 `OPENAI_API_KEY` 后启用（当前注释）
