## 一、日柱计算 — 儒略日偏移法

### 经典出处
- **《中国天文年历》**（紫金山天文台编）：儒略日定义与历法换算
- **《万年历算法详解》**（现代计算机天文历法书籍）：干支循环的数学表达

### 公式
```
JD = day + ⌊(153m + 2)/5⌋ + 365y + ⌊y/4⌋ - ⌊y/100⌋ + ⌊y/400⌋ - 32045
日柱偏移 = JD - 2415011
```

### 关键参数
- **基准点**：JD 2415011 = 1899年12月22日 = 甲子日
- **验证**：与便民查询网、参天AI、紫金山天文台历表完全一致

### 代码位置
`backend/src/lib/divination/bazi.ts` — `getDayPillar()`

## 二、年柱分界 — 立春为岁首

### 经典出处
- **《渊海子平·论天干地支》**："凡推年柱，必以立春为界，交节过后方用新年干支"
- **《三命通会·卷一》**："岁首立春，非正月朔日"

### 规则
- 公历日期在立春时刻之前：年柱用上一年干支
- 公历日期在立春时刻之后：年柱用当年干支
- **非农历正月初一，非公历1月1日**

### 代码位置
`backend/src/lib/divination/bazi.ts` — `isBeforeLiChun()`

## 六、大运排法 — 三天折合一岁

### 经典出处
- **《滴天髓·起运法》**："三天为一岁，一天为四月，一时为十日"
- **《三命通会·论大运》**：阳男阴女顺排，阴男阳女逆排

### 规则
1. 确定顺逆：阳年男命顺行 / 阴年男命逆行 / 阳年女命逆行 / 阴年女命顺行
2. 计算起运岁数：出生时刻到最近节令的天数 ÷ 3
3. 每步大运10年

## 七、八宅命卦

### 经典出处
- **《阳宅三要》**（清代赵九峰）：八宅派风水开山之作
- **《八宅明镜》**：命卦计算公式

### 公式
```
男命：(100 - 出生年后两位) % 9
女命：(出生年后两位 - 4) % 9
```
- 余数0→9
- 男命5变2（坤命），女命5变8（艮命）

## 八、真太阳时换算

### 经典出处
- **Meeus《天文算法》第二十五章**：均时差（Equation of Time）
- **《中国天文年历》**：经度时差与地方时

### 公式
```
真太阳时 = 北京时间 + 经度时差 + 均时差
经度时差 = (当地经度 - 120°) × 4分钟/度
均时差 = 9.87·sin(2B) - 7.53·cos(B) - 1.5·sin(B) 秒
其中 B = 360° × (N - 81) / 365，N为年内第几天
```

### 代码位置
`backend/src/lib/divination/solar-time.ts`

## 九、节气精确时刻

### 数据来源
- **NASA JPL DE440s 星历表**：太阳系天体位置与运动
- **Skyfield 天文库**（Python）：基于 DE440s 计算太阳黄经

### 计算方法
1. 使用 DE440s 星历表获取太阳和地球的精确位置
2. 计算太阳视黄经（apparent ecliptic longitude）
3. 当太阳黄经达到 0°, 15°, 30°, ..., 345° 时，即为 24 节气
4. 使用二分搜索精确到 0.0001°（约10秒精度）

### 精度
- 黄经精度：0.0001°（约10秒）
- 时间输出精度：分钟
- 覆盖范围：1900-2100 年

### 代码位置
- `backend/src/lib/divination/jieqi-table.ts` — 数据表
- `backend/scripts/gen_jieqi.py` — 生成脚本（使用 Skyfield + DE440s）

## 十、十神计算

### 经典出处
- **《渊海子平·论十神》**

### 规则
以日主天干为"我"，其余天干相对于我的关系：
- 同我同性 = 比肩
- 同我异性 = 劫财
- 我生同性 = 食神
- 我生异性 = 伤官
- 我克同性 = 偏财
- 我克异性 = 正财
- 克我同性 = 七杀
- 克我异性 = 正官
- 生我同性 = 偏印
- 生我异性 = 正印

### 代码位置
`backend/src/lib/divination/bazi.ts` — `calculateShiShen()`
