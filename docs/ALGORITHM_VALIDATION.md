## 一、验证方法

### 1.1 验证来源
| 来源 | 类型 | 可信度 |
|------|------|--------|
| NASA JPL DE440s 星历表 | 天文数据 | 科研级 |
| Skyfield 天文库 | 计算工具 | 科研级 |
| 便民查询网 | 在线万年历 | 高 |
| 参天AI (cantian.ai) | 专业排盘 | 高 |

### 1.2 验证维度
1. **日柱基准**：多日期儒略日计算
2. **立春分界**：边界时刻前后各测3个时间点
3. **节气分界**：每个节令当天前后测试
4. **真太阳时**：不同经度城市换算
5. **大运起运**：顺逆排计算
6. **命卦**：立春前后对比

## 四、已知异常与处理

### 4.1 无异常记录
截至 2026-04-22，所有测试用例均通过验证，无已知系统性偏差。

### 4.2 超出范围处理
当请求年份超出 1900-2100 时：
1. 年柱/命卦立春分界回退到简化算法（按日分界）
2. 月柱节气分界回退到简化算法
3. 大运起运返回默认值 3 天
4. 系统在响应中不提示精度降级（未来版本应增加提示）
